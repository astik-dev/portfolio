Privacy Policy

Last Updated: 07.04.2026

This privacy policy explains how data is collected and used when you visit this website.

Data collection and usage

No personal data is collected directly. However, the following data may be collected by the services this site relies on:

  • GitHub Pages (site hosting): visitor IP addresses are automatically logged by GitHub for security purposes, regardless of whether the visitor is signed into GitHub. This data is not accessible to this site's owner. For more information, see the GitHub Privacy Statement.
  • Umami analytics: anonymized usage data including page views, referrer URLs, browser type, operating system, device type, country of origin, and interaction events such as button clicks, link clicks, scroll events, slide navigation, and performance metrics. Analytics data is used solely to understand how visitors interact with this site.

How is data stored?

  • GitHub Pages stores server logs according to its own policies. This data is not accessible to this site's owner.
  • Umami analytics data is stored on EU-based Umami Cloud servers. All data is anonymized — no personally identifiable information is ever stored. Analytics data is retained for 6 months, after which it is automatically deleted.

Cookies

This website does not use cookies. Umami's tracking code operates without cookies and does not fingerprint or track visitors across websites.

Data protection rights

Since no personally identifiable information is collected or stored by this website, there is no personal data that can be accessed, modified, or deleted on request. For data collected by GitHub, please refer to GitHub's privacy policy and their process for data requests.

Privacy policies of other websites

This website contains links to other websites. This privacy policy applies only to this website, so if you click on a link to another website, you should read their privacy policy.

Changes to this privacy policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated date at the top.
